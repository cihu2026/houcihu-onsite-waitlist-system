# 後慈湖現場候補系統 v5

## 功能

- QR Code 現場候補登記
- 遊客自填姓名 / 電話 / 人數
- 管理後台叫號
- 手機版工作人員操作
- 電視即時叫號看板
- Google Sheets 雲端同步

## 頁面

- index.html 入口
- visitor.html 遊客首頁
- checkin.html 候補登記
- admin.html 後台
- mobile.html 手機版
- screen.html 看板
- calendar.html 營運資訊

## 部署

上傳至 GitHub Pages 即可使用

## 技術

HTML / CSS / JavaScript  
Google Apps Script  
Google Sheets
