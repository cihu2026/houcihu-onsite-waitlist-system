# 後慈湖現場補位系統

提供景區現場候補登記、依序補位、即時叫號與名額管理的數位系統。

---

## 專案特色

- 現場 QR Code 掃碼報名
- 自動產生候補號碼
- 依序遞補與叫號
- 管理後台操作介面
- 現場大螢幕叫號頁面
- 降低排隊爭議與客訴

---

## 頁面架構

| 頁面 | 功能 |
|------|------|
| index.html | 遊客現場報名 |
| admin.html | 工作人員管理後台 |
| screen.html | 現場叫號顯示畫面 |

---

## 使用技術

- HTML5
- CSS3
- JavaScript
- localStorage
- GitHub Pages

---

## 適用場景

- 景區導覽候補
- 熱門活動報名
- 名額限制入場
- 現場排隊管理
